## Get Graph

Retrieve detailed information about a specific graph by its ID.

### Path Parameters

<ParamField path="graph_id" type="string" required>
  The unique identifier of the graph
</ParamField>

### Response

<ResponseField name="graph" type="object">
  Details of the requested graph
</ResponseField>

<ResponseExample>
```json
{
  "graph": {
    "id": "graph_123abc",
    "name": "Example Graph",
    "created_at": "2023-05-01T12:00:00Z",
    "updated_at": "2023-05-02T14:30:00Z",
    "node_count": 5,
    "edge_count": 4
  }
}
```
</ResponseExample>
