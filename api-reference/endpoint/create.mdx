## Create Node

Create a new node in a specified graph.

### Path Parameters

<ParamField path="graph_id" type="string" required>
  The unique identifier of the graph
</ParamField>

### Request Body

<ParamField body="id" type="string" required>
  The unique identifier for the new node
</ParamField>

<ParamField body="type" type="string" required>
  The type of the node (e.g., "HumanActor", "AIActor")
</ParamField>

<ParamField body="properties" type="object">
  Additional properties specific to the node type
</ParamField>

### Response

<ResponseField name="node" type="object">
  Details of the created node
</ResponseField>

<RequestExample>
```json
{
  "id": "node_456def",
  "type": "AIActor",
  "properties": {
    "name": "Language Model",
    "model": "gpt-3.5-turbo"
  }
}
```
</RequestExample>

<ResponseExample>
```json
{
  "node": {
    "id": "node_456def",
    "type": "AIActor",
    "properties": {
      "name": "Language Model",
      "model": "gpt-3.5-turbo"
    },
    "created_at": "2023-05-03T10:15:00Z"
  }
}
```
</ResponseExample>
