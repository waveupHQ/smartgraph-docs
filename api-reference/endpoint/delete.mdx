## Delete Edge

Remove a specific edge from a graph.

### Path Parameters

<ParamField path="graph_id" type="string" required>
  The unique identifier of the graph
</ParamField>

<ParamField path="edge_id" type="string" required>
  The unique identifier of the edge to be deleted
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the edge was successfully deleted
</ResponseField>

<ResponseExample>
```json
{
  "success": true
}
```
</ResponseExample>
